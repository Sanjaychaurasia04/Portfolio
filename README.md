## Installation

### 1. Clone the repository

```bash
git clone https://github.com/Sanjaychaurasia04/mern-portfolio.git
cd mern-portfolio
```
2. Backend Setup
  ``` cd server
npm install
```
3. Frontend Setup
```cd ../client
npm install
npm run dev
```
